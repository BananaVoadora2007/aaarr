Os Principais Comandos para Criar um Site

Comando HTML

O HTML (HyperText Markup Language) é a linguagem fundamental para a construção de sites. Aqui está um exemplo básico de um comando HTML:


<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título do Documento</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
</body>
</html>

Comando CSS

O CSS (Cascading Style Sheets) é utilizado para estilizar o HTML. Abaixo está um exemplo básico de um comando CSS:


body {
    font-family: Arial, sans-serif;
}

h1 {
    color: blue;
    font-size: 30px;
}

h2 {
    color: red;
    font-size: 20px;
}

p {
    color: black;
    font-size: 15px;
    font-family: 'Times New Roman', serif;
}
